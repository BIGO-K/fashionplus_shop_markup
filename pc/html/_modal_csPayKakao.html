카카오페이

카카오톡 안에서 개인카드(신용/체크)를 등록하여 간단하게 결제할 수 있는 모바일 결제 서비스입니다

  • PC 결제 고객
  • 모바일 결제 고객

카카오톡에 탑재되어 별도의 어플을 설치할 필요가 없습니다

  1. 결제수단에서 카카오페이 > [결제하기] 선택

  2. 휴대폰 번호/생년월일을 입력 후 [결제요청] 선택

  3. [결제하기] 선택 후 카톡메시지 확인

  4. 카카오페이지 카드등록 및 선택 후 [결제하기] 선택

  5. [카카오페이 비밀번호 또는 지문] 입력

    1. [ARS 인증 요청] 선택 후 고객 인증(200만원 미만 결제 시 인증 페이지 미노출)

  6. 결제 인증완료 [확인] 선택

  7. 결제완료

  1. 결제수단에서 카카오페이 > [결제하기] 선택

  2. 카드등록 및 선택 후 [결제하기] 선택

  3. [카카오페이 비밀번호 또는 지문] 입력

  4. [ARS 인증 요청] 선택 후 고객 인증(200만원 미만 결제 시 인증 페이지 미노출)

  5. 결제 인증완료 [확인] 선택

  6. 결제완료
