파일 업로드mm_form-file

이미지 업로드mm_form-image

# 초기화 트리거
mm.event.dispatch([file], 'update');
페이지에 처음 접속했을 때 실행됩니다.
그 외 초기화가 필요하거나 별도로 [data-file] 요소를 생성할 경우 실행합니다.
mm.form.update(대상 또는 부모)으로 사용할 수 있습니다.

# 값 전달
mm.data.get([file], 'data-file').file: [file]의 전달 값(type object / File)입니다.
[file].file는 선택 취소했을 때 값이 사라지는 이슈가 있어 mm.data.get([file], 'data-file').file을 form data로 변환하여 사용합니다.
mm.data.get([file], 'data-file').file을 가지고 있는 요소에는 __image-on 클래스가 추가됩니다.
mm.data.get([file], 'data-file').file에는 [file].file와 FileReader로 로드한 result: data...base64, width, height, orientation 값을 추가로 포함하고 있습니다.
mm.data.get([file], 'data-file').file.result는 원본, mm.data.get([file], 'data-file').file.view.result는 미리보기 이미지와 같은 비율로 변환된 이미지 데이터가 저장됩니다. 파일 선택 전 미리보기(data-file: { _default })가 있을 때에는 name 값으로 파일명이 아닌 경로가 들어가 있고, type 값이 없습니다.
미리보기를 삭제할 경우 값은 {}입니다.

(D) 파일(이미지) 속성 값

복제 (submit에 활용)mm_form-hidden

(D) DOM 레이아웃 틀어짐을 방지하기 위해 form 요소에 복제하여 전송합니다.
form 요소에 'mm_form-hidden' 클래스를 추가하여 영역을 숨겨놓습니다.
복제할 때 cloneNode(true) 깊은 복제를 사용합니다.
복제 요소는 폼 요소(input)나 폼 요소를 감싸고 있는 컨테이너 엘리먼트(div) 모두 가능합니다.
버튼은 'submit' 타입 대신 'button' 타입을 사용합니다.
file 요소는 기본 전송 대신 mm.data.get([file], 'data-file').file 또는 mm.data.get([file], 'data-file').file를 가져와 formData를 생성하여 전송합니다.
전송이 완료되면 중복이 안되도록 form 요소의 내용을 비워줍니다.

mm.form.update();로 업데이트 합니다.
mm.form.value(element, value);로 값을 변경하면서 업데이트 합니다.

파라미터가 빈 값이면 html 전체 폼 요소를 검색하여 업데이트 합니다.

파라미터가 폼 요소의 컨테이너 요소일 경우 컨테이너 내 모든 폼 요소를 업데이트 합니다.

params array 콜백함수의 파라미터로 전달할 값

프롬프트mm.bom.prompt

mm.bom.prompt('내용', sampleCallback);

옵션이 없어도 기본으로 input:text 1종 노출됩니다.
콜백함수의 1번째 파라미터로 확인(true)/취소(false) 값을 전달합니다. 콜백함수의 2번째 파라미터로 폼요소의 value 값을 배열로 전달합니다. type 값은 input 입력요소(text, tel, number, email, search, url, password, date, month, time)와 textarea, select 입니다.
옵션을 추가하여 타이틀과 콜백함수를 호출할 수 있습니다. ex) mm.bom.prompt('내용', 콜백함수, { _title: '제목', buttoms: ['확인', '취소'], params: [콜백 파라미터], forms: [{ _type: 'tel', _placeholder: '휴대폰 번호' }, { _type: 'select', _value: 200, attribute: { 'style': 'border-width: 2px;' }, options: [{ _text: '옵션1', _value: 100 }, { _text: '옵션2', _value: 200 }] }] });

Option Type Description
콜백함수 function 확인(취소)버튼을 클릭할 때 실행할 함수
_title string 제목 변경
buttons array 변경할 버튼 이름으로 [확인, 취소] 순서로 적용
params array 콜백함수의 파라미터로 전달할 값
forms array 노출할 폼 요소
[forms]._type string 가능한 폼요소 input(text, tel, number, email, search, url, password, date, month, time), textarea, select
[forms]._value string 초기 값
[forms]._placeholder string 안내 문구
[forms]._format string _type 값이 date, month, time일 때만 사용
[forms].attribute object 요소에 추가로 적용할 속성
[forms].options array _type 값이 select일 때만 사용
[options]._text string 셀렉트 옵션 문구
[options]._value string 셀렉트 옵션 값
