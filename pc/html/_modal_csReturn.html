반품 신청 방법

  • 반품신청은 상품을 수령하신 후, 7일 이내에 가능합니다. 마이쇼핑 > 취소/반품 > 반품신청 또는 1:1문의하기에서 신청하실 수 있습니다

  • 인테리어 > 가구 제품의 경우 [마이쇼핑 > 반품신청]에서 자동 반품 처리가 어려우니 반품 시 1:1문의하기에 반품문의 주시거나 유선상 업체 연락처로 반품 신청해주시기 바랍니다

반품 진행 과정
  1. 반품신청

    고객님께서 반품을 신청한 상태

  2. 반품 진행중

    반품 신청이 접수되어 진행중 상태

  3. 상품도착

    상품이 반송지에 도착/확인중 상태

  4. 반품 완료

    반품(환불)이 완료된 상태

  • 반품신청 완료 후, 상품이 7일 이내에 반송지에 도착해야 정상적인 처리가 가능합니다

  • 만약 부득이한 사정으로 반품이 지연되는 경우는 1:1 문의하기에 문의해주세요

반품 수거

마이쇼핑에서 반품 신청이 완료되면, 해당 택배사에 자동으로 반품 수거가 예약됩니다. 따라서 고객님께서는 별도로 택배사에 수거 접수를 하지 않으셔도 됩니다

  • 판매자와 협의없이 판매자의 지정 택배사가 아닌 타업체를 이용하시는 경우는 상품이 반송처리될 수 있으며, 이때 발생된 추가 배송료는 고객님께서 부담하셔야 합니다

  • 현재 일부 상품은 자동 반품 수거 서비스가 제공되지 않으므로 '반품 수거 접수' 요청 메시지가 나올 경우, 고객님께서 직접 택배사에 반품 수거를 예약하셔야 합니다

  • 반품 신청이 접수된 후, 영업일 기준 2~3일 이내에 택배 수거 방문이 없을 시 해당 택배사로 문의해주세요

반품 배송료
  • 1.구매자가 부담하는 경우
    단순변심, 사이즈/색상 변경, 배송지 잘못 입력
  • 2.판매자가 부담하는 경우
    상품 자체의 파손/손상/오염
    판매자가 제공한 상품의 상세정보와 실제 상품의 불일치
  • 판매자의 조건부 무료배송일 경우 결제금액에서 배송료를 차감하여 환불 처리됩니다
  • 반품 시 부과되는 배송료는 환불 예정 금액에서 차감됩니다
  • 상품을 보내실 때 배송료를 동봉하여 보내실 경우 환불금액에서 차이가 발생될 수 있으니 유의하세요
반품이 불가능한 경우
  • 1. 반품신청 기간 경과
    구매자 단순변심 : 상품 수령 후, 7일 이내
    판매자가 제공한 상품의 상세정보와 실제 상품의 불일치 : 상품 수령 후 3개월 이내 또는 그 사실을 알게 된 날로부터 30일이내
  • 2. 구매자의 책임사유로 상품의 멸실 또는 훼손
    내용 확인을 위한 포장 훼손은 제외
  • 3. 구매자의 사용 또는 일부 소비에 의한 상품 가치 감소
  • 4. 시간의 경과에 의한 재판매가 곤란할 정도의 상품 가치 감소
  • 5. 복제가 가능한 상품의 포장 훼손
환불 안내

  • 반품 상품의 입고 이후, 3~5일 이내에 상품검수가 이루어집니다. 판매자와 구매자 간 반품사유에 대한 분쟁사유가 없을 시, 실제 환불처리가 진행됩니다

    결제 수단 환불방식 환불소요기간
    신용카드결제 카드승인취소 반품완료 + 7~10일
    실시간계좌이체 입금계좌로 환불 반품완료 + 1~2일
    무통장입금 등록된 환불계좌로 입금 반품완료 + 1일
    휴대폰결제 당월 전액 취소 시: 휴대폰 결제취소부분취소/익월 취소 시: 등록된 환불 계좌로 입금 반품완료 + 2일

  • 환불 소요 기간은 영업일 기준으로 토/일/공휴일은 제외

  • 신용카드 승인/취소 기간은 카드사별로 차이가 있으며, 승인취소 확인에 대한 문의는 해당 카드사로 직접 문의해주세요

  • 고객님의 단순변심에 의한 반품일 경우, 주문 시 사용한 쿠폰은 복원이 불가합니다
