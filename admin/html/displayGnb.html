코너명 3
  • GNB 노출명과 서브문구는 띄어쓰기 포함 최대 10자까지 입력 가능합니다.
  • GNB 노출명이 3자 / 서브문구가 10자 일 때, 서브문구는 GNB 노출명의 길이까지만 노출된 후 말줄임 처리됩니다.
  • 서브문구의 색상 지정 시, 프론트에서 서브문구 노출영역의 배경색이 변경됩니다.
  • GNB 영역에서 '홈'은 첫번째 순서로 고정 노출되어 나머지 GNB만 순서편집이 가능합니다.
  • 순서 편집 후 [순서편집 적용] 버튼을 클릭하면 즉시 프론트에 GNB 순서가 적용됩니다. (PC, Mobile 동일하게 적용)
  • 코너 메뉴는 최상단부터 정렬된 순서로 최대 5개의 코너만 GNB에 노출됩니다. (GNB 노출여부와 노출상태가 '노출'이며, 사용여부가 '사용'으로 설정된 코너만 노출)
  • GNB 노출명과 서브문구 입력 후 [GNB 저장] 버튼을 클릭해야 최종 저장됩니다.
