# 값 전달
$([file]).data('mm.file').files: [file]의 전달 값(type object / File)입니다.
[file].files는 선택 취소했을 때 값이 사라지는 이슈가 있어 .data('mm.file').files를 form data로 변환하여 사용합니다.
.data('mm.file').files을 가지고 있는 요소에는 __image-on 클래스가 추가됩니다.
.data('mm.file').files에는 [file].files와 FileReader로 로드한 result: data...base64, width, height, orientation 값을 추가로 포함하고 있습니다.
.data('mm.file').files.result는 원본, .data('mm.file').files.view.result는 미리보기 이미지와 같은 비율로 변환된 이미지 데이터가 저장됩니다. 파일 선택 전 미리보기(data-file: { _item })가 있을 때에는 name 값으로 파일명이 아닌 경로가 들어가 있고, type 값이 없습니다.
미리보기를 삭제할 경우 값은 {}입니다.

form 요소에 'mm_form-hidden' 클래스를 추가하여 영역을 숨겨놓습니다.
복제할 때 jQuery clone(true) 깊은 복제를 사용합니다.
복제 요소는 폼 요소($('input'))나 폼 요소를 감싸고 있는 컨테이너 엘리먼트($('element')) 모두 가능합니다.
버튼은 'submit' 타입 대신 'button' 타입을 사용합니다.
file 요소는 기본 전송 대신 .data('mm.file').files를 가져와 formData를 생성하여 전송합니다.
전송이 완료되면 중복이 안되도록 form 요소의 내용을 비워줍니다.

mm.form.update();로 업데이트 합니다.

파라미터가 빈 값이면 html 전체 폼 요소를 검색하여 업데이트 합니다.
파라미터가 폼 요소일 경우 선택한 폼 요소만 업데이트 합니다.(이 때 아래 샘플처럼 변경과 업데이트를 동시에 적용할 수 있습니다.)
파라미터가 폼 요소의 컨테이너 요소일 경우 컨테이너 내 모든 폼 요소를 업데이트 합니다.
